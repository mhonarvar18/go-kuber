# 🦍 Golang Gin Web API

A simple RESTful API built using [Gin Web Framework](https://github.com/gin-gonic/gin) in Golang.

---

## 🚀 Features

- ✅ Lightweight and fast HTTP routing
- ✅ User signup and login with JWT authentication
- ✅ RESTful API structure
- ✅ Environment-based configuration
- ✅ Ready for Docker & Kubernetes
